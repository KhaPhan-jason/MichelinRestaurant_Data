Michelin Restaurants Analysis
This project performs exploratory data analysis (EDA) on Michelin-starred restaurants worldwide, providing insights into locations, awards, cuisines, pricing, and sustainability.

⸻

Dataset
•	File: michelin_my_maps.csv
•	Key Columns: Name, Address, City, Country, Price, Cuisine, Award, GreenStar
•	Preprocessing: Removed irrelevant columns (PhoneNumber, Url, WebsiteUrl, FacilitiesAndServices, Description) and extracted City and Country from Location.

⸻

Libraries
	•	numpy
	•	pandas
	•	matplotlib
	•	seaborn

⸻

Key Analyses
	1.	General Overview
	•	Dataset shape, info, and missing values.
	•	Unique values per column.
	2.	Awards & Sustainability
	•	Count of 1, 2, 3-star restaurants.
	•	Green Star certification percentage.
  3.	Geographical Analysis
	•	Top cities and countries with Michelin restaurants.
	•	Award distribution by city/country.
  4.	Cuisine Analysis
	•	Top 10 cuisine types.
  
⸻

Insights
	•	Michelin-starred restaurants are concentrated in select cities and countries.
	•	Green Star certified restaurants are less common.
	•	French and European cuisines dominate.
	•	Top-tier (3-star) restaurants are geographically concentrated.
